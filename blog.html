Which Python version to use ?

Nowadays, there is a lot of discussion and argument as to which Python version to use. In this article, you will get to know which python version to use. As python's versions are continously upgrading, people have a misconception that the new version of python has new features which are completely different from the previous ones, which is not true. In fact, the new versions are derived from the previous python version itself. Thereby, you can choose any python version to start learning (preferably the latest one) if you are a beginner. If you are already learning Python and a new update comes, don't worry about it as it won't make a huge difference to your python journey.

Is React.Js Front-End or Back-End Library ?

React.js, developed and maintained by Facebook and a team of indivisual developers is a Front-End JavaScript Library. It is used in making web designs and is very powerful. It represents the R in MERN Development. MERN stands for MongoDb, Express, React.js and Node.js
You can use React as a Front-End library instead of HTML as it is way more powerful thann HTML and has many additional Features such as Single-Page applications, Hooks etc.
